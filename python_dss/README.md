# StabLimit - Python Implementation

Полная Python реализация программы StabLimit для расчетов динамической устойчивости энергосистем.

## Структура проекта

```
python_dss/
├── models/              # Модели данных
├── calculations/        # Модули расчетов
├── ui/                  # Пользовательский интерфейс
├── utils/               # Вспомогательные утилиты
├── rastr_operations/    # Интеграция с RASTR
├── excel_operations/    # Работа с Excel
├── visualization/       # Визуализация графиков
├── main.py              # Точка входа
└── requirements.txt     # Зависимости
```

## Функциональность

1. **Определение шунта КЗ** - расчет шунтов однофазного и двухфазного КЗ
2. **Определение предельного времени отключения КЗ** - расчет критического времени
3. **Пакетный расчет динамической устойчивости** - серия расчетов ДУ
4. **Определение МДП ДУ** - максимально допустимые перетоки мощности
5. **Определение остаточного напряжения при КЗ** - расчет на границе устойчивости

## Зависимости

- Python 3.8+
- openpyxl - работа с Excel
- matplotlib - построение графиков
- pywin32 (Windows) - интеграция с RASTR через COM
- tkinter - пользовательский интерфейс (встроен в Python)
