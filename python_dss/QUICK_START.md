# Быстрый старт Python версии StabLimit

## Установка

```bash
cd python_dss
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Основные функции

### 1. Добавление файлов
- Нажмите кнопку **"Добавить"**
- Выберите файлы:
  - `*.rst`, `*.rg2` - расчетные режимы
  - `*.scn` - аварийные процессы
  - `*.vrn` - ремонтные схемы
  - `*.ut2` - траектория утяжеления
  - `*.sch` - сечения
  - `*.kpr` - графический вывод
  - `*.csv` - задание для шунтов КЗ
  - `*.lpn`, `*.dwf` - файлы ПА

### 2. Настройка параметров
- Нажмите кнопку **"Настройки"**
- Установите необходимые параметры расчетов

### 3. Запуск расчетов
- Выберите нужный тип расчета
- Нажмите соответствующую кнопку
- Дождитесь завершения расчета

### 4. Результаты
- Результаты сохраняются в `~/StabLimit/`
- Excel файл: `Результат расчетов.xlsx`
- Графики (если включено): PNG файлы

## Примеры использования

### Расчет шунтов КЗ
1. Добавьте расчетные режимы (`*.rst`)
2. Добавьте файл задания шунтов КЗ (`*.csv`) или отметьте узлы в модели
3. Настройте параметры в "Настройки"
4. Нажмите "Определение шунта КЗ"

### Пакетный расчет ДУ
1. Добавьте расчетные режимы
2. Добавьте аварийные процессы
3. (Опционально) Добавьте файл графического вывода
4. Нажмите "Расчет ДУ"

## Требования

- Windows (для работы с RASTR)
- Установленный ПК RUSTab
- Python 3.8+
- Все зависимости из requirements.txt
