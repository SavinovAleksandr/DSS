# Руководство по декомпиляции DynStabSpace.exe

## Метод 1: ILSpy GUI (Самый простой)

### Шаги:

1. **Откройте ILSpy:**
   ```bash
   open /Applications/ILSpy.app
   ```

2. **Откройте файл:**
   - В меню: `File` → `Open`
   - Или нажмите `Cmd+O`
   - Выберите файл `DynStabSpace.exe`

3. **Просмотрите структуру:**
   - В левой панели отобразится дерево классов и методов
   - Выберите нужные файлы для просмотра

4. **Сохраните исходный код:**
   - `File` → `Save Code...`
   - Или нажмите `Cmd+Shift+S`
   - Выберите папку `source_code/`
   - Нажмите `Save`

### Преимущества:
- ✅ Простой графический интерфейс
- ✅ Возможность просмотра кода перед сохранением
- ✅ Поддержка экспорта в различные форматы

## Метод 2: ILSpy.CommandLine

### Установка:

```bash
# Установите .NET SDK (если еще не установлен)
# Затем установите ilspycmd
dotnet tool install --global ilspycmd
```

### Использование:

```bash
# Декомпиляция в папку
ilspycmd DynStabSpace.exe -o source_code/

# Декомпиляция с дополнительными опциями
ilspycmd DynStabSpace.exe -o source_code/ --language CSharp --project-file
```

### Преимущества:
- ✅ Автоматизация через скрипты
- ✅ Пакетная обработка
- ✅ Интеграция в CI/CD

## Ограничения декомпиляции

⚠️ **Важно понимать:**

1. **Имена переменных** могут быть изменены компилятором
2. **Комментарии** будут потеряны
3. **Структура кода** может отличаться от оригинала
4. **Оптимизации компилятора** могут изменить логику
5. **Obfuscated код** может быть сложно декомпилировать

## Рекомендации

1. **Сохраните оригинальный .exe** файл перед декомпиляцией
2. **Используйте версионный контроль** для отслеживания изменений
3. **Документируйте** найденные паттерны и структуры
4. **Тестируйте** декомпилированный код перед использованием

## Следующие шаги
