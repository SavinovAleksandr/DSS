## Функциональность

### ✅ Реализовано полностью:

1. **Определение шунта КЗ**
   - Однофазное и двухфазное КЗ
   - Использование отмеченных узлов
   - Типовые значения остаточного напряжения

2. **Определение предельного времени отключения КЗ**
   - Итерационный поиск критического времени
   - Настраиваемая точность

3. **Пакетный расчет динамической устойчивости**
   - Расчет с ПА и без ПА
   - Сохранение графиков
   - Обработка всех сценариев

4. **Определение МДП ДУ**
   - Расчет с учетом ПА и без
   - Утяжеление режима
   - Контроль сечений

5. **Определение остаточного напряжения при КЗ**
   - Поиск границы устойчивости
   - Итерационное смещение точки КЗ
   - Расчет остаточных напряжений

## Использование

```python
from ui.main_window import MainWindow

app = MainWindow()
app.run()
```

Или через командную строку:

```bash
python main.py
```
