# Анализ программы DynStabSpace

## Общая информация

**Название программы:** DynStabSpace (Dynamic Stability Space)
**Тип:** .NET WPF приложение (PE32+ executable, Mono/.Net assembly)
**Версия:** 1.0.0.0
**Namespace:** DynStabSpace

## Зависимости

Из анализа DLL файлов:
- **EPPlus.dll** - работа с Excel файлами
- **EPPlus.Interfaces.dll** - интерфейсы EPPlus
- **OxyPlot.dll** - построение графиков
- **OxyPlot.Wpf.dll** - интеграция OxyPlot с WPF

## Функциональность

Из анализа строк в исполняемом файле обнаружены следующие модули расчетов:

1. **UostCalc** - расчет установившихся режимов
2. **TimeKzCalc** - расчет времени короткого замыкания
3. **DynamicCalc** - динамические расчеты
4. **MdpCalc** - расчет МДП (мгновенных динамических процессов?)
5. **ShuntCalc** - расчет шунтов

## Структура проекта

Программа использует:
- WPF (Windows Presentation Foundation) для UI
- XAML для разметки интерфейса
- .NET Framework 4.0.30319

## Результаты расчетов

Программа сохраняет результаты в:
- Excel файлы (`.xlsx`) - "Результат расчетов.xlsx"
- Временные файлы (`.rst`) - "mdp_calc_tmp.rst"
- Папки с датой и временем расчета
