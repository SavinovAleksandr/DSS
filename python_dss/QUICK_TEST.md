# Быстрый запуск для тестирования интерфейса

## ✅ Готово к запуску!

Приложение можно запустить для проверки интерфейса. 

## Запуск

### Способ 1: Через main.py (рекомендуется)

```bash
cd python_dss
STABLIMIT_DISABLE_LICENSE=true python3 main.py
```

### Способ 2: Через test_ui.py

```bash
cd python_dss
STABLIMIT_DISABLE_LICENSE=true python3 test_ui.py
```

## Что будет работать

✅ **Интерфейс** - полностью функционален
✅ **UI элементы** - все кнопки, поля, меню
✅ **Валидация** - проверка данных
✅ **Настройки** - изменение параметров

## Что НЕ будет работать

❌ **RASTR операции** - требуют Windows и установленный RASTR
❌ **Расчеты** - требуют RASTR для выполнения

## Примечания

- Проверка лицензии отключена через переменную окружения
- Интерфейс работает на macOS/Linux (tkinter)
- Для полной функциональности нужен Windows с RASTR

## Остановка

Закройте окно приложения или нажмите Ctrl+C в терминале.
