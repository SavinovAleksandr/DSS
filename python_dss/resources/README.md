# Ресурсы приложения

Эта директория содержит ресурсы приложения DynStabSpace.

## Файлы

### StabLimit.png
Иконка/заставка приложения, отображаемая в заголовке окна.

- **Размер:** 1536 x 1024 пикселей
- **Формат:** PNG
- **Использование:** Иконка окна для обоих UI (tkinter и CustomTkinter)
